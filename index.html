What is a bookmarklet?

A bookmarklet is a small JavaScript program stored as a browser bookmark. When clicked, it runs code to modify a webpage, extract data, or perform quick actions.



Here's an example bookmarklet!


javascript:(function() {
    var query = window.getSelection().toString();
    if (!query) query = prompt("Enter search query:");
    if (query) window.open("https://www.google.com/search?q=" + encodeURIComponent(query));
})();
